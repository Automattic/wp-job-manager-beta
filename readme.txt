Easily update to prerelease versions of WP Job Manager for testing and development
purposes.

== Description ==

**WP Job Manager Beta Tester** allows you to try out new versions of WP Job Manager
before they are officially released.

**Use with caution, not on production sites. Beta releases may not be stable.**

After activation, you'll be able to choose an update channel:

1. Beta - Update to beta releases, RC, or stable, depending on what is newest.
2. Release Candidate - Update to RC releases or stable, depending on what is
newest.
3. Stable - No beta updates. Default WordPress behavior.

== Frequently Asked Questions ==

= Does this allow me to install multiple versions of WP Job Manager at the same
time?

No; updates will replace your currently installed version of WP Job Manger. You
can switch to any version from this plugin via the interface however.

= Where do updates come from? =

Updates are downloaded from the GitHub repository where we push betas and release candidates.

= Where can I report bugs or contribute to WP Job Manger Beta Tester? =

Bugs can be reported to the [WP Job Manger Beta Tester GitHub issue tracker](https://github.com/Automattic/wp-job-manager-beta-tester).

= Where can I report bugs or contribute to WP Job Manger? =

Join in on our [GitHub repository](https://github.com/Automattic/wp-job-manager/).
